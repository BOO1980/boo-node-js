Overview

  • TypeScript builds upon JavaScript by adding static types.
  • JavaScript is a prerequisite for learning TypeScript, as TypeScript is a superset of JavaScript.
  • TypeScript makes JavaScript better
  • TypeScript is a tool that helps catch errors early and improves code quality.
  • TypeScript is developed and maintained by Microsoft.
  • TypeScript adds some extra type related features and adds strict and static typing.
  • TypeScript you define the value type
  • TypeScript is a superset of JavaScript, which means that all valid JavaScript code is also valid TypeScript code.
  • In JavaScript if you assign a number to a string variable , it will not throw and error and it will produce a longer string where it concatenates
  • i.e "3" + 2 = "32"
  • In TypeScript if you assign a number to a string variable, it will throw an error because the types are incompatible.
  • One of the biggest benefits of TypeScript is its ability to catch errors at compile time, before they reach runtime.

Keywords

TypeScript
A programming language that is a superset of JavaScript, adding static types and other features.
JavaScript
A dynamic programming language commonly used for web development, which TypeScript builds upon.
Static Types
Types that are checked at compile time, allowing for early error detection.
Superset
A language that includes all features of another language (in this case, TypeScript includes all of JavaScript's features).
